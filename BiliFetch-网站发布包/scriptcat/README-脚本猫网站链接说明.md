# 脚本猫网站链接说明

这个文件夹里的 `BiliFetch-ScriptCat.user.js` 是给脚本猫上传的脚本。

发布网站前，它还没有真实公网网址。

发布网站后，请回到网站发布包根目录，双击：

`SET_WEBSITE_URL.bat`

输入你的网站地址，例如：

`https://yourname.github.io/BiliFetch/`

工具会自动把脚本顶部的链接改成：

- `@homepageURL`：脚本猫展示的官网链接
- `@supportURL`：脚本猫展示的反馈/支持链接
- `@downloadURL`：脚本更新下载地址
- `@updateURL`：脚本自动更新检查地址

改完以后，再把 `BiliFetch-ScriptCat.user.js` 上传到脚本猫。
