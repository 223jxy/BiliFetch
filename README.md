# BiliFetch 脚本猫发布资料

当前脚本版本：`2.3.0`

## 发布页标题

BiliFetch - B站下载助手

## 一句话简介

以脚本猫为主线的 B 站视频保存助手：自动识别页面视频，默认最高可用清晰度，支持多选内容和可选本地助手合并音视频。

## 详细介绍

BiliFetch 是一个面向普通用户的脚本猫脚本。

安装后，打开 B 站视频页，页面右下角会出现 BiliFetch 面板。脚本会自动识别当前页面的视频、分 P 或合集内容，用户可以自行勾选要下载的视频，并选择清晰度。

默认下载方式是“合并视频+音频MP4”。如果视频本身是完整 MP4，会直接下载；如果是音视频分离，脚本会优先把任务交给可选本地助手处理。本地助手用于提高下载和合并成功率，不是必装项。

## 核心功能

- 自动识别 B 站普通视频、分 P、合集。
- 用户自行勾选要下载的视频。
- 默认最高可用清晰度。
- 支持选择 1080P、720P、480P 等清晰度上限。
- 下载内容支持多选：合并 MP4、视频流、音频、字幕、弹幕、封面、信息 JSON、直链信息。
- 面板可拖动，并记住上次位置。
- 支持可选本地助手，提高音视频分离场景的成功率。
- 出错时提供中文原因和处理建议。

## 2.3.0 更新日志

## 权限说明

- `GM_download`：用于下载视频、音频、字幕、封面等文件。
- `GM_xmlhttpRequest`：用于请求 B 站接口、视频资源和本地助手。
- `GM_setClipboard`：用于复制直链、诊断信息或命令说明。
- `GM_registerMenuCommand`：用于在脚本菜单里打开面板、设置和诊断功能。
- `@connect api.bilibili.com`：读取视频信息。
- `@connect comment.bilibili.com`：读取弹幕。
- `@connect *.hdslb.com` 和 `@connect *.bilivideo.com`：访问 B 站视频、音频、封面等资源。
- `@connect 127.0.0.1`：连接用户本机可选本地助手。
- `@require ffmpeg.wasm`：本地助手未启动时，尝试浏览器内自动合并。

## 安全和合规说明

- 不绕过登录、会员、版权、地区限制。
- 只处理当前页面和 B 站接口正常返回的资源。
- 本地助手只监听 `127.0.0.1`，只允许本机浏览器访问。
- 请只下载自己有权保存和使用的内容。

## 推荐配套网站说明

网站不是主线产品，网站服务脚本猫：

- 展示脚本功能。
- 提供脚本安装入口。
- 提供更新说明和教程。
- 下载可选本地助手。

发布网站后，使用网站发布包里的 `SET_WEBSITE_URL.bat` 更新脚本中的 `@homepageURL`、`@supportURL`、`@downloadURL`、`@updateURL`，再上传到脚本猫。
